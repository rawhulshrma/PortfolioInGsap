## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js
- MongoDB

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/rayhulshrma/PortfolioInGsap.git
   ```

2. Navigate to the project directory:

   ```bash
   cd PortfolioInGsap
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the server:

   ```bash
   npm run dev
   ```
